# Digital India — Poster Showcase Website

A premium, interactive web application developed as part of a **Poster Making Competition** for the Cloud Application Developer program. This website acts as the digital companion to our physical poster. 

🎯 **Scan & Explore:** Visitors can scan a QR code placed directly on our competition poster to automatically launch this live site on their mobile devices or laptops.

## 🚀 Key Features

* **Interactive Design:** Smooth animations, custom typewriter styling, a dynamic scroll progress bar, and a responsive mobile menu.
* **Digital India Journey:** A comprehensive, scannable timeline tracking milestones from the mission's launch in 2015 up to AI-powered governance visions.
* **Core Technological Pillars:** Displays key components driving modern India, including Cloud Computing, AI, Cybersecurity, and E-Governance.
* **Responsive & Lightweight:** Built using plain HTML5 and advanced modern CSS3 (Glassmorphism, CSS variables, native grid/flexbox layouts) ensuring smooth performance across all scanning devices.

---

## 🛠️ Project Structure

* `index.html` — Single-page architecture containing semantics, custom embedded glassmorphic styling, responsive break-points, and the active JavaScript interactivity layer.

---

## 📱 How It Works (Competition Flow)

1. **The Poster:** The physical poster features a custom-generated QR code linked to this deployed project.
2. **The Scan:** Judges or visitors scan the QR code using their smartphones.
3. **The Web Landing:** The user is immediately brought to this site where they can interactively explore the deep technical stack behind the "Digital India" framework.
